# 周工作总结 Skill

## 基本信息

- **名称**: 周工作总结
- **功能**: 周工作总结生成工具 + 知识汇总
- **触发词**: "写周报"、"周工作总结"、"本周总结"
- **关键词**: Weekly report, work summary, 周报, 周工作总结, status report, progress report, work log, weekly update

---

## 功能说明

### 核心功能

1. **素材整合**: 收集一周的工作记录、日总结、新闻监测记录
2. **结构化输出**: 按照规定的5个必需内容生成周工作总结
3. **知识沉淀**: 提取可复用的知识、流程、工具
4. **知识汇总**: 汇总本周知识增量，更新知识库

### 知识汇总能力

| 汇总类型 | 汇总内容 | 更新位置 |
|---------|---------|---------|
| 项目进展汇总 | 本周各项目进展 | `项目库\项目跟踪记录.md` |
| 知识增量汇总 | 本周新增知识 | `个人知识库\学习记录.md` |
| 能力变化汇总 | 本周能力提升 | `个人知识库\能力图谱.md` |
| 关联发现汇总 | 本周发现的关联 | `项目库\_项目知识\项目关联图谱.md` |

---

## 工作流程

```
步骤1：收集一周素材
─────────────────────
工作记录\ （一周的工作碎片）
    ↓
输出成果\日总结\ （一周的日总结）
    ↓
输出成果\新闻监测记录\ （一周的新闻）
    ↓
项目库\项目跟踪记录.md
    ↓
项目库\_项目知识\ + 个人知识库\

步骤2：整合和分析
─────────────────────
所有素材 → 分类整理 → 提取关键信息

步骤3：知识汇总
─────────────────────
本周知识增量 → 汇总分析 → 更新知识库
     │
     ├── 项目进展 → 更新 项目跟踪记录.md
     ├── 知识增量 → 更新 学习记录.md
     ├── 能力变化 → 更新 能力图谱.md
     └── 关联发现 → 更新 项目关联图谱.md

步骤4：生成周工作总结
─────────────────────
关键信息 + 知识汇总结果 → 按5个必需内容组织 → 周工作总结
                                                    ↓
                            输出成果\周工作总结\MM.DD-MM.DD周工作总结.md

步骤5：用户确认和完善
─────────────────────
周总结 → 用户确认 → 补充完善

步骤6：归档
─────────────────────
周总结 + 日总结 + 新闻记录 → 输出成果\归档\2026年第X周\
```

## 附件管理规则

### 附件命名规范

**格式**：`附件N_附件名称.md`

**示例**：
- `附件1_AI辅助河西道路项目工作流程.md`
- `附件2_诺维信项目问题汇总.md`
- `附件3_背街小巷实施方案成本调整难点分析.md`

### 附件存储位置

```
输出成果\周工作总结\
├── MM.DD-MM.DD周工作总结.md      # 主文档
├── 附件1_XXX.md                   # 附件1
├── 附件2_XXX.md                   # 附件2
└── 附件3_XXX.md                   # 附件3
```

### 附件类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 工作流程 | 详细工作流程说明 | AI辅助工作流程 |
| 问题汇总 | 项目问题及解决方案 | 诺维信项目问题汇总 |
| 分析报告 | 深度分析报告 | 成本调整难点分析 |
| 工作可视化 | 详细工作梳理 | 工作可视化文档 |
| 会议纪要 | 会议详细记录 | 会议重点与待办事件 |

### 附件引用方式

```markdown

|---------|---------|------|
| 附件1 | [附件名称](./附件名称.md) | 简要说明 |
```

## 复用成果编号规则

### 编号格式

**格式**：`R-YYYY-XX-NNN`

| 部分 | 含义 | 示例 |
|------|------|------|
| R | 复用成果标识 | R |
| YYYY | 年份 | 2026 |
| XX | 周数 | 11（第11周） |
| NNN | 序号 | 001、002、003 |

**示例**：
- `R-2026-11-001`：2026年第11周第1个复用成果
- `R-2026-11-002`：2026年第11周第2个复用成果

### 复用成果类型

| 类型 | 说明 | 存储位置 |
|------|------|---------|
| 规则 | 工作规则、规范 | `.trae\rules\` 或当前文件夹 |
| 场景 | AI协作场景 | `个人知识库\学习记录.md` |
| 流程 | 工作流程 | 当前文件夹 |
| 模板 | 文档模板 | 当前文件夹 |
| 工具 | 工具使用方法 | `.trae\skills\` |

### 复用成果记录方式

