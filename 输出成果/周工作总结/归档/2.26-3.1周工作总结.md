#### A.2 治理定位

> **核心定位**：背街小巷治理不是道路整体翻修、楼房整体粉刷、绿化整体重栽，而是**修损洗污**，确保整体设施干净整齐、功能完善。

**"六治"理念**：
- 政治引领、综治能动、法治保障、德治教化、自治强基、智治支撑

#### A.5 治理内容（三类）

| 类型 | 内容 | 说明 |
|------|------|------|
| **基础类** | 便道修复、缺株少树补植、绿化井盖破损修复 | 必答题 |
| **提升类** | 口袋公园绿化、停车设施 | 群众需求 |
| **特色类** | 历史文化、可持续运营 | 差异化打造 |

#### A.6 治理原则

1. **一街一巷一策**：不搞千城一面、千篇一律
2. **群众盼什么我们干什么**：以街道为单位做好居民调查
3. **点线路相结合**：街镇是包装项目的重要依托
4. **滚动实施**：久久为功，牢固树立持久战思想
5. **简约节约**：注重功能需求

#### A.7 保障机制

**四级联动**：
- 市级：定方向、出标准、统资源、督检考
- 区级：做统筹、抓衔接，制定治理规划
- 街道：搭平台、聚合力，用好"吹哨报道""接诉即办"
- 社区：听民意、抓落地，培育自我管理能力

**专业支撑**：
- "责任设计师"制度：责任规划师+街巷设计师+运营咨询师

**资金来源**：
- 中央财政支持城市更新行动专项资金
- 超长期国债
- 市级专项基金

**道路路面病害类型**：

| 病害类型 | 说明 |
|---------|------|
| 纵横裂缝 | 缝宽10mm及以内 |
| 网裂、龟裂 | 表面或深层 |
| 车辙 | 15mm以下 |
| 井周沉陷 | 局部病害 |
| 坑槽 | 零星分布 |

**修复工艺选择**：

| 病害程度 | 修复方案 | 结构形式 |
|---------|---------|---------|
| 轻微裂缝（<10mm） | 罩面修复 | 4cm细粒式沥青混凝土 |
| 连续裂缝、网裂、龟裂 | 面层挖补 | 次干路：4cm+8cm；支路：4cm+6cm |
| 剥落、叽浆、坑槽、沉陷 | 基层补强 | 一步：+9cm ATB-30；两步：+18cm石灰粉煤灰碎石+18cm ATB-30 |

#### B.5 筛选标准

1. **符合老旧小区周边的道路**：市发改委明确要求
2. **剔除次干道**：次干道由市区级财政出资
3. **避免重复申报**：剔除已纳入城市更新项目的道路
4. **依托既有计划**：从区管道路三年/五年计划中快速梳理

#### B.6 案例价值

**成功要素**：

| 要素 | 说明 |
|------|------|
| **精准卡位** | 精准落在政策支持范围内 |
| **快速响应** | 一天之内跑通住建和城管系统 |
| **避免重复** | 提前剔除已改造的小区，形成闭环 |
| **系统思维** | 道路是"连接器"，是片区更新的切口 |

**可复用经验**：
- 区管道路既有计划是快速出量的关键依据
- 道路改造不是孤立的，是"连接器"
- 从"工程"到"片区更新"的思路
- 可作为河东老旧小区项目（P-2026-003）的参考案例

**攻防问答要点**：

| 质疑 | 应对策略 |
|------|---------|
| 时间紧如何保证质量？ | 依托既有计划，红头文件支撑 |
| 道路是否真正"衔接主干网"？ | 地图展示+服务导向论证 |
| 混合小区如何证明专款专用？ | 片区生态逻辑，老旧小区居民占比 |
| 是否为"面子工程"？ | 安全出行+无障碍设施+管网配套 |
